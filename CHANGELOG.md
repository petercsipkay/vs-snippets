# Change Log

All notable changes to the "VS Snippets" extension will be documented in this file.

## [1.0.5] - 2024-03-23

### Added

-   Enhanced license validation system
-   Support for 40+ programming languages
-   Cloud sync capabilities
-   Backup and restore functionality
-   Advanced folder organization
-   Drag and drop support
-   Search functionality
-   Tags support
-   Notes and metadata for snippets

### Changed

-   Improved UI/UX for snippet editor
-   Enhanced error handling
-   Better performance for large snippet collections

### Fixed

-   Various bug fixes and improvements

## [1.0.0] - 2024-03-15

-   Initial release of VS Snippets
-   Basic snippet management functionality
-   Folder organization
-   Code editor with syntax highlighting
