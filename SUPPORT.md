# Support

If you need help with VS Snippets, use one of these channels:

- Open a bug report or feature request: https://github.com/petercsipkay/vs-snippets/issues
- Include VS Code version, extension version, OS, and reproduction steps.

## Common checks

1. Confirm your backup folder is configured in VS Code settings (`snippets.backupFolder`).
2. Verify the folder is accessible and synced if you use Dropbox/Google Drive/OneDrive.
3. Reload VS Code after changing settings.
