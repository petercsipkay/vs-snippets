## Usage

### Managing Snippets

1. Click the Snippets Explorer icon in the Activity Bar
2. Create folders to organize your snippets
3. Add snippets to your folders
4. Edit snippets with the built-in editor

### GitHub Gist Sync

1. Click the Settings icon in the Snippets Explorer
2. Select "Manage GitHub Token"
3. Configure your GitHub Personal Access Token
4. Use Push/Pull to sync with GitHub Gists

### Import/Export

-   Export: Settings > Export Snippets
-   Import: Settings > Import Snippets

### Keyboard Shortcuts

-   Search Snippets: `Ctrl/Cmd + Shift + F`
-   Clear Search: `Esc`
-   Add Folder: `Ctrl/Cmd + Shift + N`
-   Add Snippet: Select folder, press `Ctrl/Cmd + N`

## Supported Languages

Popular languages like JavaScript, TypeScript, Python, Java, C#, and many more are supported with proper syntax highlighting and icons. The extension automatically adapts to your current VS Code theme, ensuring that:

-   Syntax highlighting matches your theme's color scheme
-   Language icons match your VS Code's file icon theme
-   UI elements follow your theme's color palette
-   Dark and light themes are fully supported

## Configuration

Access settings through:

1. VS Code Settings (`Ctrl/Cmd + ,`)
2. Search for "Snippets Manager"
3. Configure:
    - GitHub Token
    - Storage Location
    - Backup Folder
