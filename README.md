## Getting Started

1. Install the extension from VS Code Marketplace
2. Open the Snippets Manager view in the sidebar
3. Start creating folders and snippets

### Managing Snippets

1. Create folders to organize your snippets
2. Add snippets to folders
3. Edit snippets in the dedicated snippet editor
4. Add notes and specify language for syntax highlighting

### Backup and Sync

1. VS Code Settings (`Ctrl/Cmd + ,`)
2. Search for "Snippets Manager"
3. Configure:
    - Storage Location
    - Backup Folder
